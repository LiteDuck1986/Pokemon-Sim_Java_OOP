# Pokemona simulators OOP (Java)

Vienkārša Java programma, kas simulē pokemonu spēlīti, kur spēlētajs var ķert, skatīt pokemonus, arstēt savu pokemonu un cīnīties ar savu pokemonu pret pretinieku.

---

## Funkcijas
- Pokemona ķeršana
- Pokemona skatīšana un atribūtu skatīšana  
- Cīņas, kur spēlētajs var - Uzbrukt, Runāt un neko nedarīt pret AI pretinieku, kurš dara to pašu.
- AI Pretinieks, kas uz random strāda un izvēlas ko darīs savā gājienā.
